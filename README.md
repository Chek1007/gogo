# Getitle
just a weak scanner


## Usage

* -ip:
  input network segment with submask like 192.168.1.1
  
* -ports:
  the ports you want to scan, input like 81,82,83

* -threads:
  int, default is 4000
  
* -mod:
  1. straight:
    scan all the IP in your input
    
  2. smartB:
    Heuristic scanning the input but your input's submask must be 16
